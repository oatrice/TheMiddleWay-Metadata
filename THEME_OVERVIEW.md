## 📥 วิธีทดสอบ

### 🤖 Android — ดาวน์โหลด APK

1. เปิดลิงก์จากมือถือ Android:  
   👉 [**ดาวน์โหลด APK จาก GitHub**](https://github.com/oatrice/TheMiddleWay-Android/releases/latest)
2. กดที่ **`app-debug.apk`** ใน Assets
3. ติดตั้ง → อนุญาตจากแหล่งที่ไม่รู้จัก (ปกติ) → เปิดแอป

### 🍎 iOS — ผ่าน TestFlight

1. ติดตั้ง [**TestFlight**](https://apps.apple.com/app/testflight/id899247664) จาก App Store (ฟรี)
2. รอรับ **ลิงก์เชิญ** จากทีมพัฒนา → กดติดตั้ง

> ⏳ กำลังตั้งค่า — ติดตามที่ [Issue #2](https://github.com/oatrice/TheMiddleWay-IOS/issues/2)

### 🌐 Web — เปิดจาก Browser (ไม่ต้องติดตั้ง!)

👉 *(กำลัง deploy ขึ้น Vercel — จะอัปเดตลิงก์เร็วๆ นี้)*

> เปิดได้ทั้งมือถือและคอม — ติดตามที่ [Issue #5](https://github.com/oatrice/TheMiddleWay-Metadata/issues/5)

## ✅ สิ่งที่ต้องทดสอบ

### 1️⃣ เปิดแอปครั้งแรก
| ดูอะไร | ควรเห็น |
|--------|---------|
| สีพื้นหลัง | ฟ้าอ่อนสดใส |
| ชื่อแอป | "The Middle Way" สีน้ำเงินเข้ม |
| ไอคอนมุมขวาบน | 🌙 พระจันทร์เสี้ยว |
| แถบด้านล่าง | Home, Library, Courses, Profile |

### 2️⃣ กดสลับเป็นสีเข้ม
| ดูอะไร | ควรเห็น |
|--------|---------|
| สีพื้นหลัง | น้ำเงินกรมท่าเข้ม |
| ไอคอน | เปลี่ยนเป็น ☀️ ดวงอาทิตย์ |
| ปุ่ม/ลิงก์ | เป็นสีทอง |
| ตัวหนังสือ | สีขาว/อ่อน อ่านง่าย |

### 3️⃣ กดสลับกลับ
| ดูอะไร | ควรเห็น |
|--------|---------|
| สีพื้นหลัง | กลับเป็นฟ้าอ่อน |
| ไอคอน | กลับเป็น 🌙 พระจันทร์ |

### 4️⃣ ปิดแอปแล้วเปิดใหม่
| ดูอะไร | ควรเห็น |
|--------|---------|
| ธีมสี | ยังเป็นธีมเดิมที่เลือกไว้ (ไม่รีเซ็ต) |

### 5️⃣ กดดูทุกหน้า
| ดูอะไร | ควรเห็น |
|--------|---------|
| Home / Library / Courses / Profile | ทุกหน้าแสดงธีมเดียวกัน ไม่มีหน้าค้าง |

## 🐛 เจอปัญหา?

จดแล้วส่งให้ทีม:

1. **ทดสอบข้อไหน?** (เช่น ข้อ 2)
2. **ใช้เครื่องอะไร?** (เช่น Samsung S24 / iPhone 16 / Chrome)
3. **เกิดอะไรขึ้น?** (เช่น สียังไม่เปลี่ยน)
4. **ถ่ายรูปหน้าจอ** (ถ้าสะดวก)
