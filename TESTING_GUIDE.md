## 🎯 ภาพรวม

แอป "The Middle Way" รองรับ 2 ธีม:
- ☀️ **Bright Sky** (Light) — พื้นหลังสีฟ้าอ่อนสดใส, accent สีฟ้าสด
- 🌙 **Deep Cosmos** (Dark) — พื้นหลังสีน้ำเงินเข้ม, accent สีเหลืองอำพัน

ขอความเมตตาหลวงพี่ช่วยทดสอบตาม Test Cases ด้านล่าง ✨🙏

## 🔧 วิธีเปิดแอปแต่ละแพลตฟอร์ม

### 🌐 Web
```
1. เปิด Terminal
2. cd ไปที่โฟลเดอร์ Platforms/Web
3. พิมพ์: npm run dev
4. เปิด Browser → ไปที่ http://localhost:3000
```

### 🍎 iOS
```
1. เปิด Xcode
2. เปิดไฟล์ Platforms/iOS/TheMiddleWay.xcodeproj
3. เลือก Simulator (เช่น iPhone 16 Pro)
4. กดปุ่ม ▶ Run
5. แอปจะเปิดขึ้นใน Simulator
```

### 🤖 Android
```
1. เปิด Android Studio
2. เปิดโฟลเดอร์ Platforms/Android
3. รอ Gradle sync เสร็จ
4. เลือก Emulator หรือต่ออุปกรณ์จริง
5. กดปุ่ม ▶ Run
6. แอปจะเปิดขึ้นใน Emulator/อุปกรณ์
```

## 🎨 ตัวอย่างหน้าจอ (Reference)

### Light Mode — "Bright Sky" ☀️
- พื้นหลัง: สีฟ้าอ่อนสวยสดใส
- ปุ่มและ accent: สีฟ้าสด (Bright Blue #2563EB)
- การ์ด: สีฟ้าอ่อนอีกโทนหนึ่ง โปร่งใส
- ข้อความหลัก: สีน้ำเงินเข้ม
- ข้อความรอง: สีเทาอมฟ้า

### Dark Mode — "Deep Cosmos" 🌙
- พื้นหลัง: สีน้ำเงินเข้มมาก (Navy #0A192F)
- ปุ่มและ accent: สีเหลืองอำพัน (Amber #F59E0B)
- การ์ด: สี slate เข้ม
- ข้อความหลัก: สีขาว/ivory อ่อน
- ข้อความรอง: สีเทาอ่อน
